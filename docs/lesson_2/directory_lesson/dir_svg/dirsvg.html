Ефект розмиття

  • Атрибут id елемента filter визначає унікальне ім'я фільтра
  • Ефект розмиття створюється за допомогою елемента feGaussianBlur
  • Частина in = "SourceGraphic" визначає, що ефект створюється для всього елемента
  • Атрибут stdDeviation визначає розмір розмиття
  • Атрибут filter елемента rect пов'язує посиланням елемент з фільтром "f1"

Створення тіні за допомогою SVG

Фільтр feColorMatrix використовується для трансформації кольорів зміненого зображення в сторону чорного кольору. Три значення '0.2' в матриці множаться на значення червоного, зеленого і синього каналів. Зменшення їх значень змінює колір в бік чорного (значення чорного кольору = 0)
