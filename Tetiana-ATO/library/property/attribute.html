2. Підключення шрифтів за допомогою правила @ font-face.

Є альтернативний спосіб, при якому файли зі шрифтами зберігаються разом з іншими ресурсами сайту. Для них прийнято заводити окрему директорію в корені проекту - наприклад,fonts. У неї слід помістити файли для кожного зображення в потрібних форматах - в більшості випадків, якщо не потрібна підтримка старих браузерів, підійдуть .woff та .woff2.

Генератори шрифтів :

  • LocalFont
  • Transfonter

Після того, як шрифти додані в проект, їх потрібно підключити в CSS-файлі. Для цього використовується правило @font-face 

            
            @font-face {
                font-family: "Roboto";
                font-style: normal;
                font-weight: 400;
                font-display: swap;
                /*Браузер спочатку спробує знайти шрифт локально */
                src: local("Roboto"),
                / * Якщо не вийшло, завантажить woff2 * /
                url("/fonts/roboto.woff2") format("woff2"),
                / * Якщо браузер не підтримує woff2, завантажить woff * /
                url("/fonts/roboto.woff") format("woff");
              }

              / * Тепер можна використовувати шрифт * /
              body {
                font-family: "Roboto", "Arial", sans-serif;
              }
              

              Для поліпшення продуктивності правило @ font-face найкраще прописувати на самому початку CSS-файлу. 
              Так браузер зможе раніше почати обробку шрифту

Стилі, що застосовуються для оформлення тексту

  • font-family

    Даний стиль служить для завдання гарнітури шрифту. Назва шрифтів перераховуються через кому, в разі, якщо назва складається більш ніж з одне слово необхідно використовувати лапки.

    Приклад використання стилю:

    font-family: (шрифт без засічок), Arial, Helvetica, sans-serif;

    font-family: (шрифт з засічками), "Times New Roman", Times, serif;

    font-family: (моноширинний шрифт), "Roboto Mono", monospace;

  • font-size

    Даний стиль задає колір тексту. Для завдання кольору можна використовувати:

    • так і ряд ключових слів (color: red, color: black...)
    • або від 0 до 255 (в десятичній системі числення): color: rgb(51, 193, 26);

    Даний стиль задає вирівнювання тексту всередині батьківського блоку. Може мати значення left, right, center. Є ще - justify , яке вирівнює текст по всій ширині батьківського блоку.

    text-decoration:line-through. Закреслення тексту

  • letter-spacing

    Міжсимвольна відстань. Значення данної властивості вказується в одиницях довжини (пікселі, дюйми, pt), або у відносних одиницях - em.

  • word-spacing

    Властивість CSS word-spacing відрізняється від попереднього тим, що встановлює відстань між словами, а не між символами. Для даної властивості теж передбачені значення "normal" і "inherit". Можна задавати негативні значення. Тут показаний приклад запису стилю та його вигляд: p { word-spacing: 6px; }

Позиціонування елементів

  • position

    За допомогою цієї властивості визначається позиція елемента щодо браузера:

  • position:absolute

    Вказує, що елемент абсолютно позиціонується, при цьому інші елементи відображаються на веб-сторінці немов абсолютно позиціонованого елемента і немає. Положення елемента задається властивостями left, top, right і bottom, також на положення впливає значення властивості position батьківського елемента. Так, якщо у батька значення position встановлено як static або батька немає, то відлік координат ведеться від краю вікна браузера. Якщо у батька значення position задано як fixed, relative або absolute, то відлік координат ведеться від краю батьківського елементу.

  • position:relative

    Положення елемента встановлюється щодо його вихідного місця. Додавання властивостей left, top, right і bottom змінює позицію елемента і зрушує його в ту чи іншу сторону від первісного розташування.

  • position:inherit

    Успадковує значення батька

    Наприклад,

    position: absolute; left: 100px; top: 50px;right: 50px; bottom: 50px;

  • border-radius

    Закруглення кутів у CSS можна зробити для любого елемента HTML-сторінки. Для цього необхідно замінити своє значення border-radius з відповідним значенням. Часто всього значення вказується в пікселях, але можна також використовувати і інші одиниці, наприклад, em або відсотки (в останньому випадку, коли розрахунок робиться відносно ширини блоку).

    Наприклад.
    Можна зробити закруглені краї лише для верхніх або нижніх кутів (border-radius: 25px;), або задати кожному куту різний радіус скруглення

    Замість простих круглих кутів можна задати еліптичне скруглення. Для цього понадобиться вказати два значення, розділені косою лінією (для горизонтальної та вертикальної напівосі елліпсу). border-radius: 200px/100px;

    Принцип скруглення кутів. На малюнку нижче показано, яким чином обраховується скругленість кутів у CSS. Так, якщо для кута вказано одне значення, - наприклад, 20 пікселів, - це означає, що закруглення буде відбуватися по колу з радіусом 20 пікселів. У випадку, коли задаються два значення через косую лінію, наприклад, 30px / 20px, закруглення кутів буде відбуватися по еліпсу. Перше значення в такому випадку є довжиною горизонтальної напівосі еліпсу - 30 пікселів, а друге - довжина вертикальної напівосі - 20 пікселів:

background-image

Встановлює фонове зображення для елемента. Якщо одночасно для елемента задано колір фону, він буде показаний, поки фонова картинка не завантажиться повністю. Box-shadow

Можна додати до елементів тіні та змінювати їх зовнішній вигляд за допомогою CSS-властивостей box-shadow. Цей стиль здатний виконати цікаві ефекти, наприклад, об'ємність і трьоххмерність блоку.
