2.4 Пишемо 1:

 До вашого попереднього домашнього завдання додати:
- до кожної логічної частини вашої сторінки додати свій клас атрибут, який відповідно назвати англійською (цифра не може йти першою в назві класу). тобто у хедері елементи мають мати один клас. в мейні елементи мають мати інший клас. у футері - інший клас. елементи з мейну також мають мати відповідний клас, що відноситься до мейну.
тобто ми називаємо усі елементи кожної семантичної частини своїми класами (різними для кожної частини)
- до заголовків секцій, частин сторінки - додати унікальні ID
- ділимо сторінку на умовні сеції. додаємо тег section у контент header, до 1-3 блоків всередині мейну (ділимо контент по змісту) та всередині футеру (беремо весь контент у section). для кожного тегу section додаємо атрибут id="назва"
- додати у header список з навігацією по сторінці. додаємо у перелік навігації хоча б 3 елементи. навігація має подібну структуру -> nav -> ul -> li -> a. Для кожної '<'a'>' задаємо href з "#назва" - назва ID який ми додали у секціях. обов'язково використовуємо на початку #. У текст тегів '<'a'>'' пишемо назви наших секцій
